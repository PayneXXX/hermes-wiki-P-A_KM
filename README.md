# Hermes Dual-Vault Knowledge System

一套面向人类与 AI Agent 的双轨知识库体系。基于 Obsidian 管理，受启于 Karpathy's LLM Wiki。

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/PayneXXX/hermes-wiki-P-A_KM.git
cd hermes-wiki-P-A_KM

# 2. 安装到你的系统
# PKM (个人知识库) → 移到 iCloud Obsidian 目录
# AKM (Agent 知识库)  → 移到本地任意位置 + Git 管理

# 3. 在 Obsidian 中开启两个 Vault，开始使用
```

## 双库架构

| 库 | 位置 | 用途 | 管理者 |
|---|---|---|---|
| **Penn_PKM** | iCloud / 任意同步方案 | 个人阅读、灵感、阅读 | 人 |
| **Hermes_AKM** | 本地 + Git | Agent 执行、技能、操作流 | AI Agent |

详细特性请见 ""了解更多。

## 安装要求

- 至少一个支持 YAML Frontmatter 的 Markdown 编辑器（Love It: Obsidian）
- Git (用于 AKM 版本管理)
- 可选: Hermes Agent / Claude Code / OpenClaw 等 Agent 工具
